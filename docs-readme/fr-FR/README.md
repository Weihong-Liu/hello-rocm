## Qu’est-ce que hello-rocm ?

&emsp;&emsp;**hello-rocm** regroupe des tutoriels pour déployer et affiner de **grands modèles de langage sur GPU AMD avec ROCm** : environnement unifié, déploiement, fine-tuning et sujets « infra » / opérateurs.

&emsp;&emsp;Depuis **ROCm 7.10.0**, ROCm peut s’installer dans des environnements Python virtuels de façon proche de CUDA, avec prise en charge officielle de **Linux et Windows**.

## Parcours d’apprentissage recommandé

1. **[00-Environment](../../00-Environment/README.md)** — base ROCm + PyTorch + **uv**  
2. **01-Deploy** — LM Studio, vLLM, Ollama, llama.cpp, etc.  
3. **02-Fine-tune** — exemples LoRA  
4. **03-Infra** — HIP, pile logicielle et pratiques noyau
