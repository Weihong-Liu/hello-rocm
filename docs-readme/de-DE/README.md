## Was ist hello-rocm?

&emsp;&emsp;**hello-rocm** ist eine Sammlung von Tutorials zum Betrieb und Fine-Tuning von **Großen Sprachmodellen auf AMD-GPUs mit ROCm**: einheitliche Umgebung, Deployment, Fine-Tuning sowie Infra-/Operator-Themen.

&emsp;&emsp;Seit **ROCm 7.10.0** lässt sich ROCm ähnlich wie CUDA in Python-Virtualenvs installieren, mit offizieller Unterstützung für **Linux und Windows**.

## Empfohlener Lernpfad

1. **[00-Environment](../../00-Environment/README.md)** — ROCm + PyTorch + **uv** als Baseline  
2. **01-Deploy** — LM Studio, vLLM, Ollama, llama.cpp usw.  
3. **02-Fine-tune** — LoRA-Beispiele  
4. **03-Infra** — HIP, Software-Stack und Kernel-nahe Übungen
