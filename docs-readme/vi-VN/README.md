## hello-rocm là gì?

&emsp;&emsp;**hello-rocm** là bộ hướng dẫn triển khai và tinh chỉnh **mô hình ngôn ngữ lớn (LLM) trên GPU AMD với ROCm**: môi trường thống nhất, triển khai, fine-tuning và chủ đề hạ tầng / toán tử.

&emsp;&emsp;Từ **ROCm 7.10.0**, ROCm có thể cài vào môi trường ảo Python tương tự CUDA, với hỗ trợ chính thức **Linux và Windows**.

## Lộ trình học gợi ý

1. **[00-Environment](../../00-Environment/README.md)** — nền tảng ROCm + PyTorch + **uv**  
2. **01-Deploy** — LM Studio, vLLM, Ollama, llama.cpp, …  
3. **02-Fine-tune** — ví dụ LoRA  
4. **03-Infra** — HIP, stack và thực hành kernel
